# React Product Dashboard

A React application that displays product data from the DummyJSON API in a table with infinite scrolling and editable product titles.

## Features of the React Page
- Fetch products from DummyJSON API
- Display product data in a table
- Editable product title
- Infinite scrolling using IntersectionObserver
- Clean responsive UI

## Tech Stack Used
- React
- Vite
- JavaScript
- IntersectionObserver API
- Vercel (deployment)
